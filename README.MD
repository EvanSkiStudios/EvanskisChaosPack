# THE PACK REQUIRES OPTIFINE FUNCTIONS TO GET ALL THE FEATURES  
Download whatever is the latest version: https://optifine.net/downloads  
Run the Jar (Requires Java) should install to the launcher so you just select the optifine install  
You can also use a list of fabric mods to do the same effects    

# How to install (Manually)
1. Download the pack
2. Navigate to %appdata%/roaming/.minecraft
3. Drop the resource pack zip/folder into the "resourcepack" Folder (Dont unzip if zipped)  
**In minecraft, make sure its loaded at the very top, or the only resource pack applied (besides default)**
  
  
**Settings for Optifine/needed mods so the pack works as intended**  
Navigate to Options/Video settings/Quality
```
Custom Fonts: On
Custom Entity Models: On
Random Entities: On
Custom Colors: On
Custom Items: On
Custom GUIs: On
```
